<h2>mcChatRelay</h2>
mcChatRelay is a java application that relays messages from a server to discord.

<h2>Setup</h2>
To set up and get mcChatRelay working, you will need to first edit the Main.java file and set 6 values.
You would then build the bot. (I suggest using intellij idea)
